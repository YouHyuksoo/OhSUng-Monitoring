# 📍 WORD 주소 매핑 테이블

## 변환 규칙
**PC 주소 - 6000 = WORD**

---

## ⚡ 전력 데이터

| WORD | PC 주소 | 설명 |
|------|--------|------|
| 0 | D6000 | 평균전압 |
| 1 | D6001 | 평균전류 |
| 2 | D6002 | 평균전력 |
| 3 | D6003 | A상 전류 |
| 4 | D6004 | A상 전류 |
| 5 | D6005 | A상 전력 |
| 6 | D6006 | B상 전류 |
| 7 | D6007 | B상 전력 |
| 8 | D6008 | C상 전류 |
| 9 | D6009 | C상 전력 |
| 10 | D6010 | 역률 |
| 16 | D6016 | AB상 선간전압 |
| 17 | D6017 | BC상 선간전압 |
| 18 | D6018 | CA상 선간전압 |
| 32 | D6032 | 순방향 유효전력량 |
| 50 | D6050 | TOTAL 누적전력 |

## 📋 Modbus 테스트 명령어

### 전력 데이터 읽기
```bash
npm run debug:modbus 192.168.111.186 502 1 32 50
```

### 온도 현재값 읽기
```bash
npm run debug:modbus 192.168.111.186 502 1 50 51 52 53 54 55 56 57
```

### 온도 설정값 읽기
```bash
npm run debug:modbus 192.168.111.186 502 1 60 61 62 63 64 65 66 67
```

### 모든 데이터 읽기

npm run debug:modbus 192.168.111.186 502 1 32 50 50 51 52 53 54 55 56 57 60 61 62 63 64 65 66 67
